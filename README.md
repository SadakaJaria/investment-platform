# منصة مراجعة إدارة الاستثمار والمحافظ الاستثمارية

منصة تفاعلية لمراجعة مادة **إدارة الاستثمار والمحافظ الاستثمارية** — جامعة دمشق، كلية التجارة والاقتصاد (السنة الرابعة).

## المحتوى
- **٧ فصول** مع أسئلة محلولة ومتحقّق منها
- **٦ دورات امتحانية** سابقة بالترتيب الأصلي، مربوطة بالفصول
- **امتحان شامل** يسحب أسئلة عشوائية من كل الفصول
- **القصة وراء القوانين** — ٣٤ قانوناً مشروحاً بلغة بسيطة مع الجملة التي تُكتب باليد
- حلول بصيغة "رحلة الخطوات": كل حل بطاقات مرقّمة (ماذا / لماذا / الحساب)

## طريقة الاستخدام
افتح `index.html` مباشرة في المتصفح — لا يحتاج أي تثبيت. يعمل على الموبايل والكمبيوتر.

## البناء (للمطوّرين)
الملف `investment-platform.jsx` هو المصدر. لإعادة البناء:
```bash
npm install react react-dom esbuild
npx esbuild entry.jsx --bundle --minify --charset=utf8 --loader:.jsx=jsx --outfile=bundle.js
```
ثم يُدمج `bundle.js` داخل `index.html`.
